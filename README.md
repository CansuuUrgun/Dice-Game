# Dice-Game
This project is a simple and fun dice game developed as part of a Udemy "The Complete Full-Stack Web Development Bootcamp" course. 
It was created to practice DOM manipulation, event handling, and basic JavaScript logic.
Players roll the dice to generate random results, making it a great beginner-friendly project for improving coding fundamentals.
